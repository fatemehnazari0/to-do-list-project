#  Todo List App

A simple yet visually appealing **Todo List application** built with **HTML, CSS, and Vanilla JavaScript**.
This project demonstrates task management features with a clean UI.
---

##  Features

*  Add new tasks with ease
*  Assign a time to each task
*  Mark tasks as completed
*  Delete tasks individually
*  Visual status indicator for completed tasks
*  Modern and colorful UI design
*  Responsive layout

---

##  Technologies Used

* **HTML5** – structure
* **CSS3** – styling and layout
* **JavaScript (Vanilla)** – logic and interactivity

##  How to Run

1. Clone the repository
2. Open `index.html` in your browser
3. Start managing your tasks!

##  Purpose of the Project

This project was created to practice and demonstrate:

* DOM manipulation
* Event handling in JavaScript
* UI/UX design with CSS
* Building a functional front-end project without frameworks
